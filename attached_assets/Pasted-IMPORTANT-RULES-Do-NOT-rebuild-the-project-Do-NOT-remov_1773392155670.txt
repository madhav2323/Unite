NEW COLLABORATION MODEL

The system must work like this:

Admin approves users

SAVE PERMISSION SYSTEM

Users can type and edit code.

BUT

Users cannot save changes directly.

When a user clicks "Save":

Send save request to admin

Admin receives notification

Admin options:

Approve Save
Reject Save

Only after approval should the code be saved.

• code snapshot

}
