4. Core Features

Implement the following features.

4.1 Real-Time Collaborative Editing

Use Socket.IO to broadcast editor changes.

Workflow:

1 User types in editor
2 Client emits codeChange event
3 Server receives update
4 Server broadcasts update to all connected clients

Latency should be under 200ms.

Use server-authoritative synchronization model.
